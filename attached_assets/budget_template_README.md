Budget template (CSV)

File: budget_template.csv
Location: attached_assets/

Columns (header):
- ID
- VoteID
- Items
- Date (format: DD-MM-YY)
- Estimated Amount (decimal, e.g. 1000000.00)
- Vote Type (e.g. Recurrent)

Notes:
- The CSV is Excel-compatible. Open directly in Excel, or import and specify day-month-year (DMY) date format if Excel mis-parses dates.
- If your import expects different date formatting (e.g., YYYY-MM-DD), convert the Date column accordingly before importing.
- "ID" and "VoteID" are numeric identifiers; "VoteID" should match your accounting vote identifiers.

Example rows are included to match your screenshot/sample.
